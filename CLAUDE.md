## About This Repository

This is an "awesome list" - a curated collection of resources for Samsung DeX. The repository contains:

- `README.md` - The main curated list of Samsung DeX resources, organized by category
- `images/cover.jpg` - Cover image displayed in the README

## Content Structure

The README.md is organized into these sections:
- **Official Resources** - Samsung's official DeX site and guides
- **LapDocks** - Hardware to turn smartphones into laptops
- **How to...** - Tutorials for development on DeX
- **Linux on DeX** - Apps for running Linux on DeX

## Making Changes

When adding new resources:
1. Add the resource in the appropriate section using markdown link format: `[Title](URL)`
2. Keep the alphabetical ordering within each section
3. Optionally include a brief description in parentheses after the link
